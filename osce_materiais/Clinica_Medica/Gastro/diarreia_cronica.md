# Diarreia Crônica e Síndrome de Má Absorção
> Clínica Médica · Gastroenterologia | Método: Resumo Clínico

---

## Definição

**Diarreia crônica:** > 30 dias de evolução (> 3 evacuações líquidas/dia ou > 200 g/dia)

---

## Classificação: Orgânica vs Funcional

| Critério | Orgânica (sinal de alarme) | Funcional (SII, Roma IV) |
|---|---|---|
| Idade | > 45 anos | < 45 anos |
| Evolução | Curta, progressiva | Longa, estável |
| Início | Noturno (acorda paciente) | Nunca acorda |
| Perda ponderal | Sim | Não |
| Hematoquezia | Sim | Não |
| Massa palpável | Sim | Não |
| Febre | Possível | Não |
| Adenomegalia | Possível | Não |
| Estresse | — | Desencadeia/piora |

> **Regra:** qualquer sinal de alarme → colonoscopia urgente, independente da idade.

---

## Classificação por Mecanismo (Diarreia Crônica)

### 1. Osmótica
- Cessa com jejum; gap osmolar > 125; pH ≤ 5,4
- Causas: deficiência de lactase (mais comum), intolerância a sorbitol, síndrome do intestino curto

### 2. Secretória
- Não cessa com jejum; gap osmolar < 50; pode ser noturna
- Causas: tumores secretores (VIPoma, carcinoide, gastrinoma), doença de Crohn, colite colágena/linfocítica, pós-gastrectomia, ácidos biliares

### 3. Inflamatória/Exsudativa
- Fezes com sangue, pus, muco
- Causas: DII (Crohn, RCUI), colite infecciosa crônica, neoplasia colônica, isquemia

### 4. Má Absorção
- Fezes volumosas, brilhantes, fétidas (rançosas), flutuantes (esteatorreia)
- Gordura fecal > 7 g/24h

---

## Síndrome de Má Absorção (SMA)

### Fisiopatologia
Defeito na digestão ou absorção de nutrientes (gorduras, carboidratos, proteínas, vitaminas lipossolúveis)

### Apresentação Clínica
- **Fezes:** esteatorreia (gordura fecal > 7 g/24h ou Sudan III positivo)
- **Perda ponderal** + déficits nutricionais
- Déficit vitamínico: vit A (cegueira noturna), D (osteomalacia), E (neuropatia), K (coagulopatia), B12 (anemia megaloblástica)
- Anemia, hipoalbuminemia, hipocalcemia, hipoproteinemia

### Causas Principais

| Causa | Mecanismo | Diagnóstico |
|---|---|---|
| **Doença celíaca** | Enteropatia autoimune ao glúten (trigo, cevada, centeio) → atrofia vilosidades | Anti-transglutaminase IgA + biópsia duodenal (padrão ouro) |
| **Doença de Crohn** | Inflamação transmural, qualquer segmento TGI | Colonoscopia + biópsia; TC/RM enterografia |
| **Insuficiência pancreática exócrina** | ↓ lipases, proteases; fibrose cística, pancreatite crônica | Elastase fecal < 200 μg/g (baixa) |
| **Intestino curto** | < 200 cm intestino delgado remanescente (pós-ressecção) | Clínico + imagem |
| **Giardíase** | Revestimento mucosa, sem invasão | Pesquisa de Giardia nas fezes; EDA + biópsia |
| **Doença de Whipple** | Tropheryma whipplei; infiltrado macrófagos PAS+ | Biópsia intestinal + PCR |
| **SIBO** (supercrescimento bacteriano) | Proliferação bacteriana no intestino delgado | Teste respiratório H2 com lactulose/glicose |

## Pistas Clínicas para Diagnóstico Diferencial

| Achado Sistêmico | Suspeitar |
|---|---|
| **Dermatite herpetiforme** (vesículas pruriginosas nos cotovelos/joelhos) | Doença celíaca |
| **Hiperpigmentação** | Doença de Addison |
| **Macroglossia** | Amiloidose |
| **Taquicardia + tremor + sudorese** | Hipertireoidismo |
| **3 D** (dermatite + demência + diarreia) | Pelagra (deficiência de niacina) |
| **Artrite periférica + uveíte + eritema nodoso** | DII (Crohn ou RCUI) |
| **Linfadenopatia + febre + artralgia** | Doença de Whipple |

## Diarreia Farmacogênica

- ~700 medicamentos podem causar diarreia
- Atenção: metformina, antibióticos (clindamicina → C. difficile), laxantes, colchicina, AINEs, inibidores de protease
- Diarreia antibioticogênica: clindamicina (46%), amoxicilina (20%), carbapenêmicos (19%)

## Diagnóstico Laboratorial Diferencial

| Exame | Para que serve |
|---|---|
| Gordura fecal 72h (Sudan III) | Confirma esteatorreia (> 7g/24h = SMA) |
| Elastase fecal | < 200 μg/g = insuficiência pancreática exócrina |
| Anti-transglutaminase IgA (+ IgA total) | Triagem celíaca |
| Colonoscopia + biópsia | DII, colite microscópica, neoplasia |
| EDA + biópsia duodenal | Celíaca, Whipple, Giardia |
| Coprocultura + parasitológico | Infecção crônica |
| Teste respiratório H2 | SIBO, intolerância a lactose |
| PCR Clostridium difficile | Colite por C. difficile |
| TSH | Hipertireoidismo como causa de diarreia crônica |

## Alertas OSCE ⚠️

- Fezes flutuantes e brilhantes = esteatorreia = SMA — solicitar gordura fecal
- Dermatite herpetiforme = marcador cutâneo de celíaca (anti-transglutaminase IgA)
- Elastase fecal < 100 = insuficiência pancreática grave → repor enzimas pancreáticas
- Diarreia noturna que acorda o paciente = SEMPRE orgânica (investigar urgente)
- Anti-transglutaminase IgA negativo com IgA total baixo → resultado falso negativo → pedir anti-transglutaminase IgG
